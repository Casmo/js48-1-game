# js48-1-game (MS - Mad Science)
Entry for the js48-1 (http://itch.io/jam/js48-1) game jam. Thanks to [BoosterMedia](http://www.boostermedia.com/) for hosting!

# Install
`npm install`

then

`grunt`

# Play
Open `build/index.html` in your browser.
